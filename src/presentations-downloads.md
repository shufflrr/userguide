# Downloading Files and Slides

Downloads depend on file type, group roles, folder permissions, and sharing settings.

## Download files

Users can download files when their group roles and folder permissions allow it. Administrators manage download rights in **Admin > Groups**.

The **Download** role controls what file types a group can download. Administrators can allow downloads such as:

* PPTX
* PPSX
* PDF

Administrators can also deny download.

## Download one slide

Individual slides cannot be downloaded directly. Only files, such as `.pptx`, `.png`, or `.mp4`, can be downloaded.

To download one slide, save that one slide as its own `.pptx` file, then download that file.

## Shared files

When sharing a file, the sender can choose whether to allow downloads and what format is allowed. To share a file, click the three dots next to the file name in the left navigation and choose the share option.

## Add-in files

In the Shufflrr Add-in, double-click the file name to open it in Office 365 PowerPoint. Office 365 auto-saves as you work. For Shufflrr files, use the green upload icon in the Add-in to save the updated presentation back to Shufflrr.
