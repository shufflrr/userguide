# Shufflrr Security and Content Control

Shufflrr helps organizations control who can access, use, edit, share, and download presentation content. Security is managed through folder permissions, groups and roles, admin settings, authentication, workflows, and content controls.

## Folder permissions

Folder permissions control access to content in a specific folder.

1. Click the three dots next to the folder name.
2. Select **Permissions**.
3. Enter the person or group name that should have access to the folder.
4. Choose the permission level.

Permission levels:

* **View Only**: users can only view and present from the folder.
* **Read**: users can view, present, and use material in new presentations, but cannot upload to or delete files from the folder.
* **Write**: users can view, present, use, upload, delete, and edit material in the folder.
* **Full Control**: users can take any action on material in the folder, including updating permissions on the folder itself.

For **Read** or **Write**, choose whether the person or group can share presentations. **Full Control** always allows sharing. **View Only** never allows sharing.

## Groups and roles

Administrators use **Admin > Groups** to create groups, add users, and assign site-wide roles. Roles can include upload files, create presentations, edit slides, manage users, view reports, manage billing, external file sharing, access all users, create root folders, and download.

If a user belongs to multiple groups, the user has the roles from all of those groups.

## External sharing

Administrators can allow or block external sharing for the Shufflrr app in **Admin > Settings**. Folder permissions and sharing options can further control whether specific users or groups can share content.

## Authentication

Authentication options are configured in **Admin > Settings**. Shufflrr supports Shufflrr authentication, Salesforce integration, SAML Single Sign-On, and OpenID.

## Workflows

Workflows move files through stages such as creator, compliance, and presenter. Administrators define workflow stages, where files move, and who is notified.

## Content controls

Shufflrr also supports content controls such as Brand Central for approved fonts and colors, Tags for organizing and filtering content, Slide Locking for controlling edits, Slide Linking for keeping related slides together, and mandatory slides for required content.
